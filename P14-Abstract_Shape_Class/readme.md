# P14 Abstract Shape Class

## About
This practical calculates area using an abstract class template.

## What it does
It computes the area for a circle and a rectangle using their specific formulas.

## Code Explanation
- `Shape` is abstract with an `area()` method.
- `Circle` uses `3.14 * r * r`.
- `Rectangle` uses `l * b`.
- `main` prints both calculated results.
