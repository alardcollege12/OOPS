# P2 Bank Account Methods

## About
This practical shows a bank account with deposit and withdraw functions.

## What it does
It starts an account with a balance, adds money, takes money out (if possible), and prints the result.

## Code Explanation
- `BankAccount` uses a constructor to set the initial balance.
- `deposit()` adds to the balance.
- `withdraw()` only subtracts if there is enough money.
- `display()` prints account number and final balance.
