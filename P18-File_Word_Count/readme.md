# P18 File Word Count

## About
This practical reads a file and counts everything in it.

## What it does
It tells you how many lines, words, and characters are in `test.txt`.

## Code Explanation
- `BufferedReader` reads the file line by line.
- `line.split(" ")` helps count words in each line.
- `line.length()` gives the characters.
- Results are printed at the end.
