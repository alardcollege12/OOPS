# P3 Book Class

## About
This practical uses a Scanner to take book details from the user.

## What it does
It asks for a title, author, and price, then shows what you typed.

## Code Explanation
- `Scanner` is used inside the `input()` method to read data.
- `display()` prints the stored book details.
- In `main`, we create the book, call `input()`, and then `display()`.
