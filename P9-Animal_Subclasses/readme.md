# P9 Animal Subclasses

## About
This practical uses method overriding for animal sounds.

## What it does
Dogs bark and cats meow, even though they are both animals.

## Code Explanation
- `Animal` has a basic `sound()` method.
- `Dog` and `Cat` rewrite (override) the `sound()` method.
- In `main`, we call the specific sound for each object.
