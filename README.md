# Java Practical Questions

Practical 1 = Create a class Student with data members (name, rollNo, marks). Create objects and display student details.

Practical 2 = Create a class BankAccount with data members (accountNumber, balance). Use a constructor to initialize balance and create methods to deposit and withdraw money with proper balance checking.

Practical 3 = Create a class Book with attributes (title, author, price). Write a program to input and display book details.

Practical 4 = Design a class BankAccount that demonstrates encapsulation using private data members and public getter/setter methods.

Practical 5 = Create a class Employee with private fields (id, name, salary) and provide methods to access and modify the data.

Practical 6 = Create a class Student where marks cannot be set below 0 or above 100 using setter methods.

Practical 7 = Create a base class Person and a derived class Student. Demonstrate inheritance and display details.

Practical 8 = Create a class Vehicle and derive classes Car and Bike. Show how properties are inherited.

Practical 9 = Create a class Animal and derive subclasses Dog and Cat. Add methods to display sounds of animals.

Practical 10 = Create a class hierarchy for Employee and override a method calculateBonus() in subclasses. Demonstrate runtime polymorphism.

Practical 11 = Write a Java program to demonstrate method overloading by creating multiple methods with the same name but different parameters.

Practical 12 = Create a class Shape and override a method draw() in subclasses like Circle and Rectangle.

Practical 13 = Create an abstract class Vehicle with an abstract method calculateInsurance(). Implement it in a subclass Car.

Practical 14 = Create an abstract class Shape with abstract method area(). Implement it in Circle and Rectangle.

Practical 15 = Create an abstract class Employee with abstract method calculateSalary() and implement it in subclasses.

Practical 16 = Create a class Student with method setMarks(). If marks are less than 0 or greater than 100, throw an exception with message "Invalid Marks Entered". Handle it using try catch.

Practical 17 = Write a Java program to handle division by zero using try–catch.

Practical 18 = Write a Java program that reads a text file and counts the number of lines, words, and characters.

Practical 19 = Write a Java program to write user input into a file and then read and display the content.

Practical 20 = Write a Java program that creates two threads: Thread 1 prints numbers from 1 to 50, Thread 2 prints numbers from 51 to 100. Both threads should run concurrently.

Practical 21 = Create a class Rectangle with length and breadth. Use a constructor to initialize values and calculate area and perimeter.

Practical 22 = Create a class Time with hours, minutes, and seconds. Write a program to add two time objects.

Practical 23 = Create a class Account with private balance. Allow deposit and withdrawal only if the amount is valid and balance is sufficient.

Practical 24 = Create a class Product with private fields (id, name, price). Ensure price cannot be negative using setter methods.

Practical 25 = Create a class Person with name and age. Derive a class Teacher with subject and salary. Display all details.

Practical 26 = Create a class Shape with method display(). Derive Triangle and Square classes and display their properties.

Practical 27 = Write a Java program to demonstrate constructor overloading in a class Student.

Practical 28 = Create a class Calculator with overloaded methods to add: Two integers, Three integers, Two double values.

Practical 29 = Create an interface Bank with method interestRate(). Implement it in two classes SBI and HDFC.

Practical 30 = Create an interface Printable with method print(). Implement it in classes Document and Image.
