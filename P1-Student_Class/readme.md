# P1 Student Class

## About
This practical creates a `Student` class to store details and a method to print them.

## What it does
It sets a student's name, roll number, and marks, then prints them in one line.

## Code Explanation
- `Student` class has variables for name, roll number, and marks.
- `display()` method prints the values separated by spaces.
- `Main` class creates an object, assigns values, and calls `display()`.
