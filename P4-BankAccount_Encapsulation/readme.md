# P4 Bank Account Encapsulation

## About
This practical uses private variables and public methods (Encapsulation).

## What it does
It hides the account data and uses "set" and "get" methods to change and see it.

## Code Explanation
- Variables are `private` so they are hidden.
- `setBalance()` and `setAccountNumber()` are used to store data.
- `getBalance()` and `getAccountNumber()` are used to read data.
