# P13 Abstract Vehicle Class

## About
This practical uses an abstract class for vehicle insurance.

## What it does
It defines a rule that all vehicles must calculate insurance, and the `Car` class follows it.

## Code Explanation
- `abstract class Vehicle` has an empty `calculateInsurance()` rule.
- `Car` provides the actual code for that rule.
- In `main`, we create a car and call the method.
